Datatypes

There are two main types of datatypes:

  1. Primitive
    • String
    • Number
    • Boolean
    • BigInt
    • Undefined
    • Null

    2. Example:
    let str = "Greetings"      //string
    var num = 10        //number
    let i = 999999999n;      //bigint
    var bool = new Boolean(true) //boolean
    var a; console.log(a)     //undefined
    var b = null         //null

  2. Non-Primitive
    • Object

    3. Example:
    let person = {
    firstname = "Ram",
    lastname = "Patel"
    }
