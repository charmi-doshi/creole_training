Variables

They are Containers for storing data

Example:
let a = 10 //number
var langauge = "Javascript" //string

Local Scope Variables

The scope of the variable is in region where it is defined.

example:
function a(){
var x=10; //local
document.write(data);
}

Global ScopeVariables

The variable is accessible from any function.

example:
var data = 200//global
function a(){
document.write(data);
}
