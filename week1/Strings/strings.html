String-Methods

  • length method
    • It counts the length of the string.
  • slice Method
    • Lets slice javascript
  • substring Method
    • similar to slice but any index lesser than 0 is considered 0
  • substr Method
    • similar to slice but the second param is length
  • replace Method
    • This method replaces a part of the given string with another string example:Javascript is amazing
  • replaceAll Method
    • replaces all occurances from string and can use reg-exp. example:Js is amazing.Js is everywhere.Js is used in variety of applications.
  • toUpperCase Method
    • converts string to uppercase
  • toLowerCase Method
    • converts string to lowercase
  • concat Method
    • concats the strings together example : Hello
  • trim Method
    • it removes the white space from the string. example: "   Js is versatile.   "
  • trimStart Method
    • it removes the white space from the start of the string. example: "   Js is versatile.   " length=19
  • trimEnd Method
    • it removes the white space from the end of the string. example: "   Js is versatile.   " length=19
  • padStart Method
    • This method pads a string with another string from the starting. lets pad "js" with other string
  • padEnd Method
    • This method pads a string with another string from the end. lets pad "js" with other string
  • charAt Method
    • method returns the character at a specified index (position) in a string: example:Javascript
  • charCodeAt Method
    • The charCodeAt() method returns the unicode of the character at a given position in a string example:Javascript
  • split Method
    • This method splits the string into an array of sub-strings. This method returns an array. example: Js-java-ruby-c-python can be split by '-' character.
