Numbers

Types of Numbers:

  • NaN
  • Infinity
  • Hexadecimal
  • Octal

Number Methods:

  • isFinite()
    • It determines whether the given value is a finite number. isFinite(Infinity) will return false as result.
  • isInteger()
    • It determines whether the given value is an integer.
  • parseFloat()
    • It converts the given string into a floating point number.
  • parseInt()
    • It converts the given string into an integer number.
  • toExponential()
    • It returns the string that represents exponential notation of the given number.
  • toFixed()
    • It method rounds a number to a given number of digits. example:9.68759
  • toString()
    • It returns the given number in the form of string.
