Object Methods

  • Object.entries()

    method is used to return an array of a given object's own enumerable property [key, value] pairs.
    Example:const obj={
    color:"red",
    taste:"sweet",
    fruitName:{"apple","cherry","pomegranate"}
    }
    Object.entries(obj)[]

  • Object.values()

    returns an array which contains the given object's own enumerable property values, in the same order as that provided by a for...in loop.
    Example:const obj={
    color:"red",
    taste:"sweet",
    }
    Object.defineProperty(obj,"shape",{value:"round"},)
    Object.values(obj)

    As defineProperty is bydefault not-enumerable hence shape is not displayed in values result

  • Object.getPrototypeOf()


    It returns the prototype of the object given in the param. Example:const obj={
    color:"red",
    taste:"sweet",
    }


  • Object.keys()


    It creates an array containing kys of the object
    Example:const employees={
    CEO:"abc",
    secretary:"xyz",
    sales:"pqr",
    }


  • Object.assign()


    is used to copy values from one object to another.
    Example:const employees={
    CEO:"abc",
    secretary:"xyz",
    sales:"pqr",
    }
    const equipment = {
    laptop : "dell",
    mouse : "wired",
    headphones : "sony"
    }
    const junior_employee = Object.assign(employees,equipment)
    Lets assign junior_employee its equipment as well as info about his seniors.

  • Object.freeze()


    It is a method that prevents new properties from being added to it and doesnt let previous properties be modified.
    const equipment = {
    laptop : "dell",
    mouse : "wired",
    headphones : "sony"
    }

    The output will be the object because freeze method doesnt allow modification or addition
